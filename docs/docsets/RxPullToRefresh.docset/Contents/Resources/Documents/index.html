RxPullToRefresh

A Swift library enables you to create a pull to refreshable UIScrollView with a custom view supporting RxSwift.

Requirements

  • iOS 9.0 or later
  • Swift 4.2

Installation

Carthage

Add the following to your Cartfile and follow these instructions.

github "gumob/RxPullToRefresh"

Do not forget to include RxSwift.framework. Otherwise it will fail to build the application.

CocoaPods

To integrate RxPullToRefresh into your project, add the following to your Podfile.

platform :ios, '9.3'
use_frameworks!

pod 'RxPullToRefresh'
